# 弹弹play for mac

[弹弹play](http://www.dandanplay.com/)的mac版 

## 功能介绍
1. 支持本地json、xml格式弹幕
2. 支持弹幕云过滤、支持正则表达式
3. 视频智能识别
4. 支持acfun、bilibili的弹幕搜索

## 系统要求
最低支持 osx 10.9
